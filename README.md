# IronPulse Fitness

A responsive gym website with a hero section, about, programs, trainers, pricing, testimonials, and a contact form.

## Project Structure

```
gym website/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── assets/
│   ├── images/
│   └── icons/
└── package.json
```

## Local Development

```bash
npm install
npm start
```

This starts a live-reload dev server at `http://localhost:5500`.

Alternatively, just open `index.html` directly in a browser — no build step is required.
